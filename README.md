## 1. 品类定义与对标参考 / Category Definition & Benchmark Reference

QNLOO-SPL 属于全新的**主权价值协议（Sovereign Value Protocol）** 品类，与以 Apache 2.0 为代表的传统开源协议并行，分别对应不同的产业阶段与核心诉求：
- 传统开源协议解决「代码要不要开放、如何共享」的问题，核心是技术协作效率；
- 主权价值协议解决「价值要不要对齐、如何按主权规则分配」的问题，核心是生态治理与价值主权。

QNLOO-SPL belongs to the new **Sovereign Value Protocol** category, parallel to traditional open-source licenses represented by Apache 2.0, corresponding to different industrial stages and core demands:
- Traditional open-source licenses address "whether and how to share source code", focusing on technical collaboration efficiency;
- Sovereign value protocols address "whether and how to align value under sovereign rules", focusing on ecological governance and value sovereignty.

Taking the most widely used Apache 2.0 as a benchmark, the core differences are as follows:

| :--- | :--- | :--- |
| **核心定位**<br>**Core Positioning** | 代码共享许可，保障技术自由流通<br>Code sharing license, ensuring free circulation of technology | 主权商事价值分配范式，保障价值按主权规则流转<br>Sovereign commercial value distribution paradigm, ensuring value circulation per sovereign rules |
